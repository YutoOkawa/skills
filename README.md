# Skills Repository

このリポジトリは、Google Antigravity CLI (`agy`) 向けのカスタムスキルを管理するリポジトリです。

## インストール方法

### 1. APM (Agent Package Manager) によるインストール（推奨・他端末での利用）

他端末などへの配布・インストールには、APMを利用するのが最も簡単です。

#### A. 全てのスキルを一括インストールする（ルート参照）
リポジトリのルートを指定してインストールします。これにより、すべてのスキルがまとめて導入されます。
```bash
apm install <GitHubのユーザー名>/<リポジトリ名>#v1.0.0
```

#### B. 特定のスキルのみを個別にインストールする（サブディレクトリ参照）
必要な特定のスキルのみをインストールしたい場合は、対象のサブディレクトリを指定してインストールします。
```bash
apm install <GitHubのユーザー名>/<リポジトリ名>/skills/discord-messenger#v1.0.0
```



## 開発ガイド

* **新しいスキルの追加**:
  1. `skills/` ディレクトリの下に新しいディレクトリ（例: `skills/my-new-skill`）を作成します。
  2. その中に `SKILL.md` を作成し、スクリプト等への参照は絶対パスではなく **相対パス**（例: `[my_script.py](scripts/my_script.py)`）で記述します。
  3. 新しいスキルディレクトリに移動し、`apm init` コマンドを実行して `apm.yml` を作成します。
* **一括インストールの更新**:
  新しく追加したスキルをルートからの一括インストール対象に含めたい場合は、ルートディレクトリの `apm.yml` の `dependencies.apm` セクションに相対パス（例: `- ./skills/my-new-skill`）を追記してください。
